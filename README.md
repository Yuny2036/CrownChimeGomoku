# 크라운과 차임의 오목
연습을 위한 개인프로젝트이자 니케 팬-메이드 게임. 로컬 네트워크 오목.

## 여는 말
니케 캐릭터들의 SD 이미지를 봤을 때, 저는 "전부 동그라미로 되어있네, 아이콘 같은걸로 쓰기 좋을 듯?" 라고 생각했습니다.  
그래서 작은 팬-메이드 게임을 만들기로 맘 먹었어요. 이게 그거구요. 오목이에요! \\^0 ^\/

## 설명
로컬 네트워크만 있습니다. 한 명이 서버가 되어 세션을 만들고, 다른 한 명이 서버 IP를 입력해 입장합니다.
규칙은 오목하고 같습니다: 가로, 세로 혹은 대각선 한줄로 돌 5개를 이어둡니다.
크라운과 차임이 자기 돌을 놓을 때마다 말을 합니다.

## 맺음 말
기술적, 시간적 여유가 없었다보니 오류와 결함들을 미처 손보지 못했습니다. '경기 끝났는데 왜 타이틀로 못 감?' 이나 '방화벽 문제 해결 안했음?' 같은거요. 미안해요..!

# CrownChimeGomoku
Toy project for practice, Nikke fan-made game. Local-network Gomoku.

## Opening words
When I saw SD Images for Nikke's characters, I thought something like "All the SDs was drawn in circular shape. looks great to be used as Icons or something."  
So I planned making a small, fan-made game. And this is the game: Gomoku \\^ 0^\/

## Description
Local-network only: One creates a session as a server; the other enter via typing the server's IP.
Rules are same as Gomoku: match five stones in a horizontal, vertical or diagonal line.  
Crown and Chime speak a line whenever they put their stone.

## Closing words
I had little skills and time to make and develop, so I couldn't fix some errors and flaws like 'Can't return to the title after the end of the match.' or 'You didn't managed firewall problems.' Sorry..!
